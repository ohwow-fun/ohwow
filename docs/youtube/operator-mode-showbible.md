# Show Bible — Operator Mode

> **Status:** skeleton. Run `node scripts/yt-experiments/yt-series-brief.mjs operator-mode`
> to fill this doc with deep-research findings.

## One-liner

Practical AI systems for real businesses. Show, don't tell. One specific
workflow replacing a known pain. Operator to operator, no pitch energy.

## Voice

Peer showing you the trick they use every day. Warm, direct, experienced.
Expects you to know terms. Doesn't lecture. Doesn't overpromise.

## Format contract

**Length:** 60s default. 3 min walkthroughs. 8 min full teardowns.

**Structure (60s — 3 scenes):**

1. **Pain hook (0–4s)** — name the problem in a concrete way: *"Your
   sales team spends 4 hours a day copy-pasting from Linkedin. Stop."*
2. **System reveal (4–45s)** — the workflow, in 2–3 concrete steps. Real
   tool names. Real handoff points. Real numbers where available.
3. **Outcome + next step (45–60s)** — what this gets you back. One-line
   next-step: *"Start with the lead-scoring step — that's where 80% of
   the time is trapped."*

## Source rules

Primary: SMB use-case knowledge bank (tagged `operator-mode`) — curated
from case studies, post-mortems, customer interviews, and operator posts.
Secondary: x-intel `bucket='hacks'` rows.

Numbers must be citeable or flagged as "from our experience." Tools named
must be real and currently in use somewhere. Never promise "10x" or "50%"
without a source.

## Banned

- Vague advice ("use AI to save time")
- Sales energy ("this changes everything")
- Unnamed tools ("we use a great CRM")
- Hypothetical results ("could save you hours")
- "Please" in CTAs
- OHWOW self-references in the system reveal (the soft CTA lives in the
  description, not the narration)

## Success metric

Inbound leads + site clicks + qualified viewers (views from operators
vs. general tech audience — measured via UTM on description link).

## Visual template

`workflow-steps` block (existing) + `terminal-log` for tool invocations
where the workflow includes a CLI moment. Montserrat body, green accent,
tight spacing — business feel without corporate sterility.
