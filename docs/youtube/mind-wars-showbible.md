# Show Bible — Mind Wars

> **Status:** skeleton. Run `node scripts/yt-experiments/yt-series-brief.mjs mind-wars`
> to fill this doc with deep-research findings.

## One-liner

Premium debate clips. Two positions on one question about AI, work,
meaning, or governance. Steelman both; let the viewer decide. Make them
think.

## Voice

Thoughtful moderator. Measured, curious, precise. Hold the tension
between opposing ideas without taking sides. Quote sources; attribute
arguments; never sneer.

## Format contract

**Length:** 60s clips + 8–20 min full episodes.

**Structure (60s clip — 4 scenes, split-screen friendly):**

1. **Question card (0–3s)** — a crisp question: *"Should AI own property?"*
2. **Position A (3–22s)** — the case for, in one minute's worth of real
   argument. Name the view; name the argument pattern; name a proponent
   where possible.
3. **Position B (22–45s)** — the case against, same rigor. Must *defeat*
   A on A's own terms to earn the slot.
4. **Open-ended close (45–60s)** — no verdict. The moderator asks one
   question that makes the viewer think. Comments are the scorecard.

## Source rules

Primary: knowledge-base items tagged `mind-wars` (philosophy corpus +
economics papers + founder/operator interviews). Fallback: deep-research
agent for topic scouting.

Cite real thinkers. When staging a debate between AI and a historical
figure (Nietzsche, Keynes), quote their actual positions from the corpus;
don't invent.

## Banned

- Weak strawmen ("some people say technology is bad")
- Moderator editorializing ("the right answer is…")
- Clickbait questions without a real argument
- "AI says…" voiceovers without naming a model + prompt
- OHWOW self-references

## Success metric

Watch time + saves + comment depth (avg comment ≥80 chars signals real
engagement).
