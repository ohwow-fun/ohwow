# Symbiosis — Collaboration Intelligence

**Layer 6: How humans and agents work together.**

Beyond "agents work, humans approve." Symbiosis models the evolving partnership between humans and AI.

## Philosophy

- **Aristotle (Philia)**: partnership based on mutual benefit and genuine care.
- **Confucius (Li)**: propriety — knowing when human judgment is needed vs when the agent should act.

## Modules

| Module | What it does |
|--------|-------------|
| `trust-dynamics.ts` | Per-domain trust that evolves from outcomes. 5 successes = trust rises. 1 failure = trust drops. Not static autonomy levels. |
| `handoff-intelligence.ts` | Decides when to hand off vs act autonomously. High trust + high load = skip approval. Low trust = always require. |
| `collaboration-rhythm.ts` | Detects optimal pattern per pair: delegation, review, pair, autonomous. |
| `mutual-learning.ts` | Tracks bidirectional teaching: agent output used without edits (agent taught human) vs human revisions (human taught agent). |
